## 代码管理

- 新功能开发：基于 master --> git checkout -b feature-xxx
- bug 修复：基于 master --> git checkout -b hotfix-xxx

## 项目运行

- npm install
- 本地开发： npm run dev
- 打包：npm run build
- 本地预览：npm run serve
